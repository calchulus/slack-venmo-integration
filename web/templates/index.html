Welcome to Slackmo


A Venmo integration for Slack that
lets you pay your co-workers with ease.

View Code on Github

Application details:

  • This is a sample app to showcase a Venmo slash command integration for Slack.
  • The commands:
    • /slackmo [@username] [$amount] [note] . For example, /slackmo @mauerbac $10 lunch!
    • /slackmo help
    • /slackmo [@username] add [phone #] - Used for adding a new co-worker
  • Right now, this isn't on the Slack platform, so you can test it out as a custom command on your own team. Check out the code on Github.
